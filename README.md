# racktables-plugins-japanese-navigation

RackTablesの画面で使われている文字を一部日本語化するプラグインです。
変数を上書きしているだけですので、タイトルやタブの文字にしか対応できません。

<h2>画面イメージ</h2>

<h2>インストール方法</h2>

1. plugins ディレクトリに jp_navigation.php をコピーします。

<h2>アンインストール方法</h2>

1. plugins/jp_navigation.php ファイルを削除します。
